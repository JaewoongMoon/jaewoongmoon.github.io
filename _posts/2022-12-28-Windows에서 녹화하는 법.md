# 개요
- Windows 10부터는 기본 녹화 기능이 들어가 있다. 
- Windows + G를 누르면 녹화 윈도우가 나타난다. 
- 동그라미가 그려져 있는 녹화버튼을 누르면 녹화가 시작된다. (Windows + Alt + R 을 눌러도 된다)
- 녹화가 끝나면 동그라미 버튼을 한번 더 누른다. 
- C:\Users\{유저명}\Videos\Captures 에 녹화된 비디오가 저장된다.
