# WebRTC란?
- WebRTC(Web Real-Time Communication)은 웹 애플리케이션과 사이트가 **서버와 같은 중간자 없이** 브라우저 간에 오디오나 영상 미디어를 포착하고 마음대로 스트림할 뿐 아니라, 임의의 데이터도 교환할 수 있도록 하는 기술이다. 
- 플러그인이나 제 3자 소프트웨어 설치 없이 종단 간 데이터 공유와 화상 회의를 가능하게 한다.
